# Phase 1: Feel — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Stepover feel premium by adding haptic/sound feedback, skeleton loaders, empty states, error recovery, settings cleanup, and hiding deferred modes.

**Architecture:** Six independent workstreams that can be implemented in any order. Three new shared components (`SkeletonComponent`, `EmptyStateComponent`, `ErrorStateComponent`) and one new core service (`FeedbackService`). All follow the existing standalone component + signal-based reactivity patterns.

**Tech Stack:** Angular 20 (standalone components, signals), Capacitor 7 (`@capacitor/haptics`), Web Audio API, CSS animations

**Spec:** `docs/superpowers/specs/2026-04-06-pre-launch-ux-design.md` — Phase 1 sections 1.1–1.6

Create `frontend/src/app/core/feedback.service.ts`:

```typescript
import { Injectable, signal } from '@angular/core';
import { Haptics, ImpactStyle, NotificationType } from '@capacitor/haptics';

export type SoundName =
  | 'correct'
  | 'wrong'
  | 'tick'
  | 'timeout'
  | 'achievement'
  | 'streak'
  | 'matchFound'
  | 'victory';

interface SpriteEntry {
  start: number;
  duration: number;
}

/**
 * Sprite map: start time (seconds) and duration for each sound in the sprite file.
 * Update these values to match the actual audio sprite once created.
 */
const SPRITE_MAP: Record<SoundName, SpriteEntry> = {
  correct:      { start: 0.0, duration: 0.5 },
  wrong:        { start: 0.6, duration: 0.4 },
  tick:         { start: 1.1, duration: 0.2 },
  timeout:      { start: 1.4, duration: 0.6 },
  achievement:  { start: 2.1, duration: 1.5 },
  streak:       { start: 3.7, duration: 1.2 },
  matchFound:   { start: 5.0, duration: 0.8 },
  victory:      { start: 5.9, duration: 1.5 },
};

@Injectable({ providedIn: 'root' })
export class FeedbackService {
  readonly soundEnabled = signal(this.readBool('stepover_sound', true));
  readonly hapticEnabled = signal(this.readBool('stepover_haptic', true));

  private audioCtx: AudioContext | null = null;
  private spriteBuffer: AudioBuffer | null = null;

  /** Toggle sound and persist */
  toggleSound(): void {
    const next = !this.soundEnabled();
    this.soundEnabled.set(next);
    this.writeBool('stepover_sound', next);
  }

  /** Toggle haptic and persist */
  toggleHaptic(): void {
    const next = !this.hapticEnabled();
    this.hapticEnabled.set(next);
    this.writeBool('stepover_haptic', next);
  }

  /** Light tap — answer selection, button press */
  tapLight(): void {
    this.haptic(() => Haptics.impact({ style: ImpactStyle.Light }));
  }

  /** Medium impact — correct answer, match found */
  tapMedium(): void {
    this.haptic(() => Haptics.impact({ style: ImpactStyle.Medium }));
  }

  /** Heavy impact — achievement, victory, tier promotion */
  tapHeavy(): void {
    this.haptic(() => Haptics.impact({ style: ImpactStyle.Heavy }));
  }

  /** Error notification — wrong answer, timeout */
  tapError(): void {
    this.haptic(() => Haptics.notification({ type: NotificationType.Error }));
  }

  /** Success notification — correct answer with emphasis */
  tapSuccess(): void {
    this.haptic(() => Haptics.notification({ type: NotificationType.Success }));
  }

  /** Warning notification — timer running low */
  tapWarning(): void {
    this.haptic(() => Haptics.notification({ type: NotificationType.Warning }));
  }

  /** Play a named sound from the sprite */
  playSound(name: SoundName): void {
    if (!this.soundEnabled()) return;
    const entry = SPRITE_MAP[name];
    if (!entry) return;
    this.ensureAudioCtx().then((ctx) => {
      if (!this.spriteBuffer || !ctx) return;
      const source = ctx.createBufferSource();
      source.buffer = this.spriteBuffer;
      source.connect(ctx.destination);
      source.start(0, entry.start, entry.duration);
    });
  }

  /** Combined haptic + sound for common interactions */
  correctAnswer(): void {
    this.tapSuccess();
    this.playSound('correct');
  }

  wrongAnswer(): void {
    this.tapError();
    this.playSound('wrong');
  }

  timerTick(): void {
    this.tapWarning();
    this.playSound('tick');
  }

  timerExpired(): void {
    this.tapError();
    this.playSound('timeout');
  }

  achievementUnlock(): void {
    this.tapHeavy();
    this.playSound('achievement');
  }

  streakMilestone(): void {
    this.tapHeavy();
    this.playSound('streak');
  }

  matchFound(): void {
    this.tapMedium();
    this.playSound('matchFound');
  }

  victory(): void {
    this.tapHeavy();
    this.playSound('victory');
  }

  private haptic(fn: () => Promise<void>): void {
    if (!this.hapticEnabled()) return;
    fn().catch(() => { /* Haptics unavailable on web */ });
  }

  private async ensureAudioCtx(): Promise<AudioContext | null> {
    if (this.audioCtx && this.spriteBuffer) return this.audioCtx;
    try {
      this.audioCtx = new AudioContext();
      const response = await fetch('/assets/audio/sfx-sprite.webm');
      const arrayBuffer = await response.arrayBuffer();
      this.spriteBuffer = await this.audioCtx.decodeAudioData(arrayBuffer);
      return this.audioCtx;
    } catch {
      return null;
    }
  }

  private readBool(key: string, fallback: boolean): boolean {
    try {
      const stored = localStorage.getItem(key);
      return stored !== null ? stored === 'true' : fallback;
    } catch {
      return fallback;
    }
  }

  private writeBool(key: string, value: boolean): void {
    try {
      localStorage.setItem(key, String(value));
    } catch { /* storage unavailable */ }
  }
}
```

@Component({

      <div
        class="skeleton skeleton--{{ variant() }}"
        [style.width]="width()"
        [style.height]="height()"
        [style.border-radius]="variant() === 'circle' ? '50%' : '8px'"

  styleUrl: './skeleton.css',
  changeDetection: ChangeDetectionStrategy.OnPush,

  width = input('100%');
  height = input('16px');
  count = input(1);

  items = () => Array.from({ length: this.count() });
}

```typescript
import { ChangeDetectionStrategy, Component, input, output } from '@angular/core';
import { Router } from '@angular/router';
import { inject } from '@angular/core';

@Component({
  selector: 'app-empty-state',
  standalone: true,
  templateUrl: './empty-state.html',
  styleUrl: './empty-state.css',
  changeDetection: ChangeDetectionStrategy.OnPush,
})
export class EmptyStateComponent {
  icon = input('info');
  title = input('Nothing here yet');
  subtitle = input('');
  ctaLabel = input('');
  ctaRoute = input('');
  ctaClick = output<void>();

  private router = inject(Router);

  onCta(): void {
    const route = this.ctaRoute();
    if (route) {
      this.router.navigate([route]);
    } else {
      this.ctaClick.emit();
    }
  }
}
```

## Task 13: Improve Error Interceptor with Retry Support

**Files:**
- Modify: `frontend/src/app/core/error.interceptor.ts`

- [ ] **Step 1: Add retry logic to error interceptor**

Replace the contents of `error.interceptor.ts` with:

```typescript
import { HttpInterceptorFn, HttpErrorResponse } from '@angular/common/http';
import { inject } from '@angular/core';
import { catchError, throwError, retry, timer } from 'rxjs';
import { ToastService } from './toast.service';
import { AuthModalService } from './auth-modal.service';
import { ProService } from './pro.service';

export const errorInterceptor: HttpInterceptorFn = (req, next) => {
  const toast = inject(ToastService);
  const authModal = inject(AuthModalService);
  const pro = inject(ProService);

  return next(req).pipe(
    retry({
      count: 2,
      delay: (error: HttpErrorResponse, retryCount: number) => {
        // Only retry on network errors (status 0) or server errors (5xx)
        if (error.status === 0 || error.status >= 500) {
          return timer(Math.pow(2, retryCount) * 1000);
        }
        // Don't retry client errors (4xx)
        return throwError(() => error);
      },
    }),
    catchError((err: HttpErrorResponse) => {
      if (err.status === 401) {
        authModal.open();
      } else if (err.status === 402) {
        pro.showUpgradeModal.set(true);
      } else if (err.status === 0) {
        toast.show('Connection lost. Check your network.', 'error');
      } else if (err.status >= 500) {
        toast.show('Something went wrong. Please try again.', 'error');
      }
      return throwError(() => err);
    }),
  );
};
```

This adds automatic retry (2 attempts with exponential backoff) for network and server errors before showing the error toast. Client errors (401, 402, 4xx) are not retried.
